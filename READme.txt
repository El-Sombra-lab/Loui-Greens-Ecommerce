# Loui Greens | Fresh Organic Produce

This is a PHP-based eCommerce website for Loui Greens, offering fresh organic produce. The website allows users to browse products, register, add items to the cart, and make purchases.

## Features

- Home page with a hero section displaying a welcome message and a video background.
- Navigation bar for easy navigation to different sections of the website.
- Search functionality to search for specific products.
- Cart functionality to add products and view the total price.
- Featured products section showcasing the freshest vegetables, fruits, beverages, and herbs.
- Products section displaying a list of available products with their details.
- Sidebar with delivery brands and categories for easy filtering of products.
- Footer section with additional information and links.

## Prerequisites

To run the website locally, you'll need the following:

- PHP (version 5.6 or higher)
- MySQL database
- Web server (e.g., Apache)

## Installation and Setup

1. Clone the repository or download the code files.
2. Set up a local web server environment with PHP and MySQL.
3. Import the included database file (`database.sql`) into your MySQL server.
4. Update the database connection details in the `includes/connect.php` file.
5. Place the code files in your web server's document root directory.
6. Access the website through your web browser by visiting `http://localhost`.

## Dependencies

The website uses the following dependencies:

- Bootstrap (v4.0.0) - CSS framework for responsive design.
- Font Awesome (v6.4.0) - Icon set for visual enhancements.

These dependencies are included via CDN links, so an internet connection is required for them to load properly.
